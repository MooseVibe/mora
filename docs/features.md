# Features · Mora

## Готово (MVP)

### Авторизация
- Google OAuth
- Обычная авторизация (email/пароль)
- Сессия сохраняется

### Карта дня (неавторизованный)
- Можно вытянуть карту без регистрации
- Карта показывается с описанием
- Кнопка «Сохранить карту» предлагает залогиниться
- Кнопка «Поделиться»

### Карта дня (авторизованный)
- Карта вытягивается раз в день
- Защита от повторного вытягивания — нельзя вытянуть второй раз в тот же день
- Карта автоматически сохраняется в дневник
- В колоде 28 карт: 22 старших аркана и 6 пилотных младших/придворных карт
- Уже вытянутую карту дня можно развернуть в full-result экран и вернуть назад shared-element анимацией

### Дневник карт (в личном кабинете)
- История всех вытянутых карт по дням
- Просмотр карты + описания, которое было в день вытягивания
- Навигация по датам

### Сводка по 3 последним картам (на дашборде)
- Показываются 3 карты за последние 3 дня
- Подпись «summary» под ними — в MVP **захардкожена**
- В будущем — генерация AI

## Планируется (после редизайна)

### Большое — AI-таролог в чате
Главная фича проекта. Пользователь задаёт вопрос → AI делает расклад → объясняет карты → ведёт диалог. Прикручивается после редизайна.

### Среднее
- Замена захардкоженной «сводки 3 карт» на реальную AI-генерацию
- Уведомления о новой карте дня (push / email)
- Платная подписка (детали не проработаны)

### Маленькое
- Анимация переворота карты
- Шаринг карты в соцсети с красивой картинкой
- Тёмная/светлая тема (если решим что нужно — пока тёмная одна)

## Не делаем (явно)

- Метрики и аналитика — рано, сначала продукт.
- Команда, шаринг карт между пользователями — это не социалка.
- Расклады на нескольких картах вручную — это ломает фокус на AI-таролога.
- Покупка отдельных карт, NFT, gamification — это не суть продукта.
